üöÄ Getting Started

Q: What is Validify?
A: Validify is a Web3 Discord security platform that prevents server compromises through multi-signature verification and real-time threat monitoring. We protect your community from the #1 attack vector: compromised admin accounts posting fake mint links.
Q: How does multi-signature link verification work?
A: Instead of allowing any single admin to post links, Validify requires multiple admins (e.g., 3 out of 5) to approve a link within a time window (30-60 seconds). Each admin receives a unique code via DM and must approve before the link posts. This prevents a single compromised account from posting phishing links.
Q: Is Validify free?
A: Currently in beta testing phase on Sepolia testnet. Pricing tiers:
  • Basic: 500 VLFY/month (~$50)
  • Advanced: 1,500 VLFY/month (~$150)
  • Pro: 5,000 VLFY/month (~$500)
Early beta testers get discounted rates!
Q: What blockchain is Validify on?
A: Currently deployed on Sepolia Testnet (Ethereum). Mainnet deployment planned for Q2 2026 after thorough testing and security audits.

üîê Security & Privacy

Q: Will Validify ever ask for my seed phrase or private keys?
A: NEVER. Validify only uses wallet signature verification through MetaMask. We never ask for, store, or have access to your private keys or seed phrases. If anyone claiming to be from Validify asks for this information, it's a scam.
Q: What data does Validify collect?
A: We follow a minimum data collection principle:
  • Wallet addresses (for subscription verification)
  • Discord server audit logs (for security monitoring)
  • Security event data (flagged threats, timestamps)
We do NOT collect:
  • Private messages (except when users opt-in for DM scanning)
  • Personal identifying information
  • Private keys or passwords
See our Trust & Safety page for full details.
Q: Can Validify access my Discord DMs?
A: Only if you explicitly opt-in to DM phishing protection (Phase 2 feature). This is completely optional and requires user consent. Without opt-in, Validify has zero access to your DMs.
Q: Is the smart contract audited?
A: Currently on testnet (no audit required). Before mainnet deployment, we will conduct professional security audits from firms like OpenZeppelin or Trail of Bits. Audit reports will be publicly published.

üÜò Support & Troubleshooting

Q: MetaMask isn't connecting. What do I do?
A: Common fixes:
  1. Make sure you're on Sepolia Testnet in MetaMask
  2. Refresh the page and try again
  3. Check browser console (F12) for errors
  4. Try a different browser (Chrome recommended)
  5. Update MetaMask extension to latest version
Q: Transaction failed. Why?
A: Most common reasons:
  • Insufficient Sepolia ETH: Get more from a faucet
  • Insufficient VLFY balance: Claim from faucet first
  • Not approved: Click "Activate" button again, it will request approval
  • Network congestion: Wait and try again
